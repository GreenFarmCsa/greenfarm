## 1. Introduction

Green Farm is based on CSA (Community Supported Agriculture) philosophy, devoted to motivating green agricultural production and sustainable living.

## 2. Getting Started


### 2.1. Installation

NPM

```shell
npm install
```

### 2.2. Run

```shell
npm run dev
```

### 3.2. Build

```bash
# For Stage Environment
npm run build:stage

# For Production Environment
npm run build:prod
```
